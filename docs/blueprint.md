# **App Name**: Nevta Digital

## Core Features:

- User Authentication: Secure user registration and login using mobile number and password.
- Nevta Record Creation: Create new Nevta records with occasion details.
- Guest Entry: Add guest contributions with name, location, and amount.
- Real-time Data Table: Display all entries in a table format.
- Summary Calculation: Automatically calculate and display total entries and collected amount.
- Data Synchronization: Enable offline data storage with automatic cloud synchronization.
- Report Generation: Generate and export reports in PDF and CSV formats, complete with summary details and guest-wise entries, including date & time, enabling easy sharing, archival and analysis.

## Style Guidelines:

- Primary color: A vibrant, warm gold (#D4AF37) to evoke prosperity and celebration, reflecting the cultural significance of Nevta.
- Background color: A light beige (#F5F5DC), providing a soft, neutral backdrop that ensures readability and elegance without being distracting.
- Accent color: A muted olive green (#808000) which subtly complements the gold, suggesting growth, harmony, and respect for tradition.
- Body and headline font: 'Alegreya', a serif font that gives an elegant, intellectual, contemporary feel.
- Use clear and simple icons for navigation and actions, ensuring ease of use for all users.
- Clean and intuitive layout with clear sections for data entry, display, and reporting.
- Subtle animations for data updates and report generation, providing visual feedback to the user.
